# dictation-to-shell

Dictate your shell commands on a Mac and run them directly in your terminal!

## Installation

```sh
pip install dictation-to-shell
```

## Usage

```sh
poetry run dictation_to_shell
```

## Development

It's recommended to use a virtual environment.

```sh
pip install poetry
poetry install
```

## Publish to PyPi

```sh
poetry build
poetry publish
```
